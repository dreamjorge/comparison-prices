# Wireframe notes

## Status
- Drafted (text-only). Needs low-fidelity wireframe export to PDF/images.

## Home (list)
- Header: app name + zone chip
- Primary list: items with qty, unit, last price
- CTA: floating Add button
- Secondary: Compare button (primary)
- Banner ad at bottom (Free only)
- Empty state: illustration + 'Add your first item'

## Compare (totals by store)
- Header: list name + date
- Cards: store name, total, savings vs next
- Sort: cheapest first
- Tap store -> optional per-store breakdown
- Banner ad at bottom (Free only)

## Product detail
- Title: product name
- Subheader: preferred unit
- History: 7-day chart
- Stores list: current price per store
- Substitutes (P1): grouped below
- No banner ads

## Settings
- Zone selector
- Store toggles (enabled stores)
- Notifications: price drops, best store
- Manage Pro
- Banner ad at bottom (Free only)

## Paywall
- Value props: save money, alerts, no ads
- Plans: monthly/annual
- CTA: Start Pro
- Restore purchase
- Close

## UX rules
- No interstitials
- Rewarded only for: 24h full history, multi-store compare
- Paywall gates: history > 7 days, ad-free
- Keep Add/Edit flow ad-free
